# 메모리

> Value type (복사)
    
    - struct, enum, tuple, String, Array, Dictionary 등
    원본이 한개가 있으면 그 값을 복사해 복사한 값으로 각각 사용
```swift
//Value type (복사)
var score1 = 100
var scoreA = score1
var scoreB = scoreA

scoreA = 50
/*
score1 = 100
scoreA = 50
scoreB = 100
*/
```
   
> Reference type (참조)
   
    - calss, func, closure
     원본이 한개가 있으면 그 하나를 가지고 여러곳에서 같이 사용
```swift
//Reference type (참조)
var score1 = 100
var scoreA = score1
var scoreB = scoreA

scoreA = 50
/*
score1 = 100
scoreA = 50
scoreB = 50
*/
```
